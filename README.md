# 보듬교육학원 성적 결과 생성기

보듬교육학원 학생들의 성적 결과를 A4 PDF 형식으로 자동 생성하는 도구입니다.

## 📋 기능

- **자동 데이터 파싱**: 간단한 형식으로 입력한 성적 데이터를 자동으로 분석
- **스마트 정렬**: 전교 등수, 점수, 등급 등을 기준으로 자동 정렬
- **A4 PDF 생성**: 인쇄 가능한 전문적인 A4 성적표 생성
- **실시간 미리보기**: 데이터 입력 시 즉시 결과 확인
- **온라인 배포**: Vercel을 통한 빠른 접근

## 🚀 배포

이 프로젝트는 Vercel에 배포되어 있으며 다음과 같은 이점이 있습니다:

- ✅ 별도의 설치 불필요
- ✅ 브라우저에서 바로 사용 가능
- ✅ 자동 HTTPS 지원
- ✅ CDN을 통한 빠른 로딩
- ✅ GitHub와 자동 동기화

**배포 주소:** https://bdscore-one.vercel.app/

## 📝 사용 방법

### 데이터 입력 형식

각 항목을 `---` 로 구분하여 입력합니다:

```
과목 학교 초성 점수결과
```

### 예시

```
영어 과천여고1 ㅈㅇㄹ 100점
수학 과천중앙고1 ㅈㅎㅈ 1등급
영어 과천여고1 ㅈㅇㄹ 전교 1등 (100점·1등급)
영어 과천외고1 영어과 ㅊㅇㅂ 50점 상승
수학(확통) 과천고3 *ㅁ* 40점 상승
```

### 정렬 순서 (자동)

1. 전교 5등 이내
2. 100점
3. 1등급
4. 전교 5등 이후 (등수 순)
5. 99점 → 98점 → ... → 90점
6. 2등급
7. 점수 상승 (높은순)
